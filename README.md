# Student Performance Data Analysis

This project explores factors that influence student academic performance.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## Dataset
Portuguese student dataset from UCI Machine Learning Repository.

## Key Questions
- Does study time affect grades?
- Does alcohol consumption affect performance?
- Do absences impact final grades?

## Key Insights
- Study time positively impacts student performance.
- Higher absences lead to lower grades.
- Lifestyle factors like alcohol consumption influence results
